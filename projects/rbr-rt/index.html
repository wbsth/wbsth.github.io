RBR Replay Tools

WPF Desktop app

Application that provides Richard Burns Rally simulator users some additional features.

Users can load binary replay files generated by RBR, then:

  • Extract car setup
  • Replace car setup with another one

Technologies used:

  • C#
  • WPF
  • Adonis UI
