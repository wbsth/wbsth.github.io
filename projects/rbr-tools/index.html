RBR Tools

Vue3 Web App

Web app that contains various tools for Richard Burns Rally simulator.

Main feature is online telemetry analyzer that allows users to visualize and compare data coming from sim.

Technologies used:

  • Vue 3
  • Tailwind
  • Pinia (state management)
  • Papaparse (telemetry file parsing)
  • Gridstack (interactive charts layout)
  • Highcharts (plotting charts)
