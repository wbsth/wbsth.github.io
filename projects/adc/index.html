Air Data Computer

This group project involved designing and manufacturing Air Data Computer, that would be able to measure:

  • Altitude
  • Indicated Air Speed
  • True Air Speed
  • Vertical Speed
  • Air Temperature

Designed ADC would be connected to PC using serial connection. Then, using special software, data would be recorded and plotted.

I was the person responsible for the development of the software.

App was written using C# and Winforms. It is capable of:

  • displaying measurements (using graphical gauges)
  • displaying graphs
  • saving received data to a .csv file
