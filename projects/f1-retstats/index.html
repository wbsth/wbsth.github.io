Formula 1 cars retirements data analysis

Data analysis

In this project I've analysed data from all Formula 1 races from 1950 until 2019, and plotted few interesting graphs about F1 cars reliability.

My direct inspiraction was Sebastian Vettel reaction after his Russia 2019 retirement due to engine failure

I've downloaded data from unofficial F1 API using requests and json library. Then, using bokeh, scipy, pandas I've created few interactive graphs.
