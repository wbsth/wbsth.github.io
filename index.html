home / projects / Air Data Computer

Air Data Computer

This group project involved designing and manufacturing Air Data Computer, that would be able to measure:

  • Altitude
  • Indicated Air Speed
  • True Air Speed
  • Vertical Speed
  • Air Temperature

Designed ADC would be connected to PC using serial connection. Then, using special software, data would be recorded and plotted.

I was the person responsible for the development of the software.

App was written using C# and Winforms. It is capable of:

  • displaying measurements (using graphical gauges)
  • displaying graphs
  • saving received data to a .csv file

home / projects / Glider flight recorder

Glider flight recorder

Main goals

Project involved designing and manufacturing prototype of glider flight recorder with reduced separation detection system.
Main device goals were to:

  • record glider flight data (position, altitude, speed, etc.)
  • communicate with other devices using Open Glider Network (OGN)
  • warn pilot about potential reduced separation (using sound clues)

Hardware

I've designed two side PCB using Eagle software.


Then, ABS case that fits designed board was prepared using Fusion360.


Renders you can see above were made using Keyshot.

Software

I've programmed the STM32F103C8T6 microcontroller the device is using, with STM32CubeIDE + Hardware Abstraction Layer libraries.

home / projects / On-bike car detection system

On-bike car detection system

Project involved designing and manufacturing prototype of on-bike car detection system. Two devices were used: ESP32 board with distance sensor, and a Android phone.

Main goals

  • detect distance to passing cars using ultrasonic sensor
  • raise alarm when measured distance was lower than threshold
  • make a car photo using Android device, using BT connection to trigger camera

Hardware

I've used ESP32 board with HC-SR04 ultrasonic sensor.

Software

I've made an Android app that connects to the ESP32. When ESP32 sends proper signal, app makes photo using phone camera.

home / projects / F1 cars reliability analysis

Formula 1 cars reliability analysis

Github page

In this project I've analysed data from all Formula 1 races from 1950 until 2019, and plotted few interesting graphs about F1 cars reliability.

My direct inspiraction was Sebastian Vettel reaction after his Russia 2019 retirement due to engire failure.

I've downloaded data from unofficial F1 API using requests and json library. Then, using bokeh, scipy, pandas I've created few interactive graphs.

Interactive version available here:

  • Mechanical retirements
  • Overall retirements
  • Accidents retirements
